# Инструкция по организации проекта

## Выполните следующие команды в Git Bash или CMD:

```bash
# 1. Перейдите в директорию проекта
cd "D:\Проекты проги\utmKA-2.0-2"

# 2. Запустите скрипт организации
python cleanup_and_organize.py
```

Или просто запустите `organize.bat` двойным кликом.

## Что делает скрипт:

1. ✅ Удаляет ненужные файлы (app.zip, build/, dist/, __pycache__)
2. ✅ Создает папки scripts/ и config/
3. ✅ Перемещает скрипты сборки в scripts/
4. ✅ Перемещает конфигурационные файлы в config/
5. ✅ Создает git коммит

## После выполнения:

```bash
# Проверьте результат
git status

# Если нужно, отправьте на GitHub
git push origin main
```
