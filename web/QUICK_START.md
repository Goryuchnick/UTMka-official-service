# Быстрый старт для тестирования

## Установка зависимостей

```bash
cd web
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/macOS
source venv/bin/activate

pip install -r requirements.txt
```

## Создание .env файла

Создайте файл `.env` в папке `web/`:

```env
FLASK_APP=run.py
FLASK_ENV=development
SECRET_KEY=dev-secret-key-for-testing
JWT_SECRET_KEY=dev-jwt-secret-for-testing
DATABASE_URL=sqlite:///utmka_dev.db
```

## Запуск приложения

```bash
python run.py
```

Приложение будет доступно по адресу: http://127.0.0.1:5000

## Что работает сейчас

✅ **Фронтенд полностью готов:**
- Генератор UTM-меток (работает без авторизации)
- Модальные окна авторизации
- UI элементы для подписки
- Интеграция с API

⚠️ **Backend API (заглушки):**
- `/auth/login` - принимает любой email/password, возвращает токен
- `/auth/register` - принимает данные, возвращает успех
- `/auth/me` - возвращает тестового пользователя
- `/api/v1/history` - возвращает пустой список
- `/api/v1/templates` - возвращает пустой список
- `/api/v1/subscription/status` - возвращает free план

## Тестирование

1. Откройте http://127.0.0.1:5000
2. Попробуйте сгенерировать UTM-ссылку (работает без авторизации)
3. Нажмите "Войти" и введите любой email/password
4. После "входа" попробуйте сохранить ссылку в историю
5. Увидите модальное окно "Требуется подписка"
